# Slicky/TinyJS

TinyJS parser.

TinyJS is **one line only** javascript-like language with some syntax sugar on top of it.
This makes it a good option for example for templates' expressions.

**See tests for examples**

## Installation

```
$ npm install @slicky/tiny-js
```
